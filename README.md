# Сайт филиала ветеранов ГСВГ и ГВВД — Мангистауская область

Статический адаптивный сайт. Чтобы посмотреть его, откройте `index.html` двойным щелчком в браузере.

Перед публикацией замените в `index.html` все пометки «Добавьте ...» на реальные даты, имя председателя, адрес, номер телефона, ссылки на документы и новости. В `script.js` форма сейчас показывает сообщение; для приёма обращений её нужно подключить к почте, Telegram или серверной части.
